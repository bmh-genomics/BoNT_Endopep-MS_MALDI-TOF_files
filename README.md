The Endopep-MS assay for botulinum neurotoxin (BoNT) activity was first developed at the US Centres of Disease Control (Barr JR, et al. Botulinum neurotoxin detection 
and differentiation by mass spectrometry. Emerg Infect Dis. 2005 Oct;11(10):1578-83. doi: 10.3201/eid1110.041279) and has been adapted for use on Bruker Biotyper MALDI-TOF platforms for testing various sample types and BoNT types/subtypes 
(several publications such as: Kalb SR, et al. Extraction of BoNT/A, /B, /E, and /F with a Single, High Affinity Monoclonal Antibody for Detection of Botulinum Neurotoxin by Endopep-MS. PLOS 2010. doi.org/10.1371/journal.pone.0012237)

This repository contains the 5 custom files used by Health Canada (Food & Nutrition Directorate) to perform the BoNT Endopeop-MS analysis:AutoXMethods (.axe) file; MassControlList (.mcl) file; flexAnalaysisMethods (.FAMSMethod) file; flexControlMethods (.par) file; ProcessingParameter (.prp) file

These files/settings have been validated on the Bruker MicroFLEX Biotyper system (2015-2026) and the Bruker Sirius Bioryper system (2026) and use the Peptide Calibration Standard II (with associated .mcl file).
Note: Adjustment to laser intensity will be required for the AutoXMethods (.axe).
